## 📖 Overview | نظرة عامة
**Age In Detail** is a precision-focused Python tool designed to break down a person's age into multiple time units. It provides a granular view of lived time, from months down to the very last minute.

**عن المشروع:** أداة برمجية بسيطة وفعالة مكتوبة بلغة بايثون، تقوم بتحليل عمر الإنسان وتحويله إلى وحدات زمنية تفصيلية (شهور، أسابيع، أيام، ساعات، دقائق) بدقة عالية.

---

## ✨ Key Features | المميزات
* 🎯 **Precise Calculation:** Breaks down years into 5 different time units.
* ⚡ **Instant Response:** Fast execution with zero latency.
* 💻 **CLI Interface:** Clean and straightforward Command Line interface.
* 🛠️ **Lightweight:** No external libraries required (Pure Python).
